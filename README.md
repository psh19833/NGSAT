# NGSAT (New Generation Stock Auto Trader)

> 한국 주식 자동매매 시스템 — SAT3 후속 차세대 프로젝트

## 프로젝트 개요
NGSAT은 SAT3(StockAutoTrader-V3)의 구조적·전략적 한계를 모두 극복하는 완전히 새로운 자동매매 시스템입니다.

## 핵심 특징
- **3단계 매매 파이프라인**: 시장 레짐 평가 → 기술지표/차트패턴 종목 선별 → ML 매매 결정
- **완전 모듈화**: 한 모듈 = 한 책임, 모듈 간 결합도 최소화
- **실거래/백테스트 완전 분리**: 같은 저장소 내 별도 패키지, 상호 import 금지
- **모든 매매 판단에 근거 명시**: 근거 없는 거래 결정 금지
- **다중 증권사 지원**: KIS 시작, 어댑터 패턴, 추후 구독 서비스 확장
- **대시보드 + 텔레그램**: 웹 UI와 모바일 알림 동시 지원

## 기술 스택
- 언어: Python 3.12+

- ML: scikit-learn → XGBoost/LightGBM (예정)
- 알림: Telegram Bot
- 패키지 관리: uv
- 프론트엔드: React + Vite + TailwindCSS (모던 다크 테마)
