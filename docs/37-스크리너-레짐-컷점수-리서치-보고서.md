| 레짐 | 권장 컷 | 통과율 | 100종목 예상 | 최대 선정 | 예상 후보수 |

### 3.3 현행 vs 권장 비교


## 4. 권장 실행 방안

### 4.1 DB 설정 업데이트 (ConfigService)

```bash
curl -X POST http://localhost:8000/api/control/config \
  -H "Content-Type: application/json" \
  -d '{
    "screener_bull_min_score": 55,
    "screener_neutral_min_score": 60,
    "screener_bear_min_score": 65
  }'
```

### 4.2 코드 기본값 변경 (선택사항)

`core/config.py:120-124`:
```python
screener_bull_min_score: float = 55.0  # 60.0→55.0
screener_neutral_min_score: float = 60.0  # 35.0→60.0
screener_bear_min_score: float = 65.0  # 50.0→65.0
```

환경변수 기본값도 동일하게 설정:
```python
# core/config.py:260-264
s.screener_bull_min_score = float(os.getenv("NGSAT_SCREENER_BULL_MIN_SCORE", "55.0"))
s.screener_neutral_min_score = float(os.getenv("NGSAT_SCREENER_NEUTRAL_MIN_SCORE", "60.0"))
s.screener_bear_min_score = float(os.getenv("NGSAT_SCREENER_BEAR_MIN_SCORE", "65.0"))
```

### 4.3 DB 현행 ConfigService 값 정리

현재 DB에 저장된 값 중 코드 기본값과 다른 것만 ConfigService에 남기고,  
**동일한 것은 삭제하는 것이 좋습니다** (혼란 방지):
