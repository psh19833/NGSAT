### A-2. Orchestrator DB — 세션 팩토리 패턴 전환 (High, 30분)
- **파일**: `live/orchestrator.py`
- **현황**: 단일 `Session(self._db_engine)` 인스턴스를 보관하여 비동기 코루틴 간 공유 위험
- **영향**: 경쟁 조건으로 인한 예외 또는 데이터 손상 가능성
- **처리**: 세션 인스턴스 대신 `sessionmaker` 팩토리를 보관하고 각 코루틴이 `with Session.begin():` 으로 독립 세션 사용

```python
# AS-IS
from sqlalchemy import create_engine
from sqlalchemy.orm import Session
self._db_engine = create_engine(...)
self._db_session = Session(self._db_engine)

# TO-BE
from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker
self._db_engine = create_engine(...)
self._Session = sessionmaker(bind=self._db_engine)
# 사용 시: with self._Session() as session: ...
```
