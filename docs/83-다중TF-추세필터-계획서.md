---

## 1. 개요

현재 NGSAT는 **일봉 + 분봉** 2개 타임프레임만 사용 중.  
장기 추세 방향(주봉)이 빠져 있어, 일봉에서 좋아 보이지만 주봉에서 하락 추세인 종목에 진입할 위험이 있음.

다중 TF 필터: **주봉(장기 추세) + 일봉(중기 모멘텀) + 분봉(실시간 타이밍)** 방향이 모두 일치할 때만 최종 매수 승인.

---

## 2. 현재 상태

| 타임프레임 | 현재 사용 | 용도 |
|---|---|---|
| 주봉(weekly) | ❌ 없음 | — |
| 일봉(daily) | ✅ ML 예측 + 스크리닝 지표 | 가격, RSI, MACD, ADX, OBV 등 |
| 분봉(minute) | ✅ 단타 진입/청산 정밀화 | 30bars 기반 예측 |

```python
def _check_multitf_alignment(self, code, daily_prices, weekly_prices) -> MTFFilterResult:
    """
    3개 TF 방향성 일치도 검사.

    Returns:
        aligned: True/False
        reason: 불일치 사유
        score: 0~1 일치도
    """
    # 1. 주봉: MA5 > MA20 (상승 추세)
    weekly_ma5 = sma(weekly_closes, 5)
    weekly_ma20 = sma(weekly_closes, 20)
    weekly_up = weekly_closes[-1] > weekly_ma5[-1] > weekly_ma20[-1]

    # 2. 일봉: MACD > 0 (모멘텀)
    daily_macd = macd_line[-1] > signal_line[-1]

    # 3. 분봉: 최근 5분봉 상승 중
    minute_up = len(minute_prices) >= 5 and minute_prices[-1].close > minute_prices[-5].close

    score = sum([weekly_up, daily_up, minute_up]) / 3
    aligned = score >= 0.67  # 3개 중 2개 이상 일치
    return MTFFilterResult(aligned=aligned, score=round(score, 2), ...)
```

## 4. 리스크 평가

| 항목 | 리스크 | 내용 |
|---|---|---|
| **주봉 데이터 수집** | 🟢 낮음 | KIS API `FID_PERIOD_DIV_CODE="W"`만 변경. 부트스트래핑 시 1회 |
| **Rate Limit 증가** | 🟡 보통 | 유니버스 100종목 × 주봉 1회 = 100회 추가 API 호출. 초기 부팅 시에만 |
| **필터 강제성** | 🟢 낮음 | aligned=false면 ML 예측 skip. 진입 기회를 놓칠 수 있으나 이게 목적 |
| **기존 로직 영향** | 🟢 없음 | `_evaluate_and_execute` 루프 내 continue만 추가. 기존 흐름 변경 없음 |
| **분봉 데이터 없을 때** | 🟢 안전 | minute_prices=None이면 분봉 조건 생략, 주봉+일봉 2개로만 판단 |

**총평: 🟢 리스크 낮음. 진행 가능.**
