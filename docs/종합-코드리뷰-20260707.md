| 항목 | 내용 |
|------|------|
| **파일** | `live/risk.py:151` |
| **문제** | KIS API `daily_loss_pct`가 음수(-3.5 = 3.5% 손실)일 때 `loss_pct >= limit_pct` 비교가 항상 False → **일일 손실 한도가 절대 트리거되지 않음** |
