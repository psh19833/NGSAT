## 2. 추가 전략 (논문/전문가/퀀트 기반)

### 2.1 섹터 로테이션 전략

**출처**: Journal of Portfolio Management, "Sector Rotation Strategies"  
**원리**: 하락장에서도 방어적 섹터(헬스케어, 필수소비재, 유틸리티)는 상대적으로 선방

| 섹터 | 약세장 평균 수익률 | 특징 |
|:-----|:-----------------:|:------|
| 헬스케어/제약 | -2~+3% | 경기 방어적, 필수 수요 |
| 필수소비재 | -1~+2% | 경기 둔감 |
| 유틸리티 | 0~+3% | 배당 + 방어 |
| 통신 | -2~+1% | 안정적 현금흐름 |

**구현**: `screener.py`에 섹터별 스코어 가중치 추가 (BEAR 시 방어 섹터 +5~10점)

### 2.2 헤지 전략 — 인버스/KODEX 인버스 활용

**출처**: Ray Dalio (All Weather Portfolio), 퀀트 헤지펀드  
**원리**: KOSPI 하락 시 인버스 ETF로 헤징 또는 단기 수익

```
KOSPI -7% → KODEX 인버스(곱버스) +3~+7%
```

**한국 상품**:
- KODEX 200선물인버스 (하락 시 수익)
- KODEX 200선물인버스2X (하락 시 2배 수익)
- TIGER 미국S&P500선물(H) 인버스

**리스크**: 인버스는 기간에 따른 decay 존재 (3일 이상 보유 시 contango 비용)

### 2.3 변동성 기반 포지션 사이징 (Kelly Criterion)

**출처**: Kelly (1956), Thorp (1969), "The Kelly Capital Growth Investment Criterion"  
**원리**: 레짐 점수와 ATR에 따라 포지션 크기를 동적 조절

```
현재: BEAR = 고정 0% (HOLD)
제안: BEAR 점수에 따라 0~8% 동적 배분
  - BEAR 35점 = 8%
  - BEAR 20점 = 4%
  - BEAR 5점 = 2%
```

**구현**: `risk.py`에 `fractional_kelly(regime_score, atr)` 함수 추가

### 2.4 Mean Reversion + Pair Trading

**출처**: Gatev, Goetzmann & Rouwenhorst (2006, Journal of Finance)  
**원리**: 하락장에서 과매도된 종목과 상대적으로 선방한 종목 간 스프레드 회귀

```
pair = (삼성전자, SK하이닉스)
삼성전자 -10%, SK하이닉스 -3% → 스프레드 확대
→ 삼성전자 매수 + SK하이닉스 매도 (스프레드 회귀 베팅)
```

**한계**: 한국 주식 공매도 제한 (2024년 6월 이후 부분 재개). 페어 트레이딩 불가.

### 2.5 VIX/TVIX 유사 변동성 전략

**출처**: Whaley (2000), "The Investor Fear Gauge"  
**원리**: KOSPI 200 변동성 지수(VKOSPI)가 급등할 때 매수

```
KOSPI 급락 → VKOSPI 급등 → 변동성 매수
→ 시장 안정 → VKOSPI 하락 → 수익
```

**한국 상품**: KODEX 변동성 (VKOSPI 추종)
**한계**: 장기 보유 시 decay, 단기 트레이딩 필요

### 2.6 모멘텀-가치 혼합 (Dual Momentum)

**출처**: Gary Antonacci (2014), "Dual Momentum"  
**원리**: 절대 모멘텀(현재 > 12개월 전) + 상대 모멘텀(강한 종목)

```
BEAR: 절대 모멘텀 음수 → 현금 비중 확대
BUT: 상대 모멘텀(업종 내 상위)인 종목은 선별 매수
```

**구현**: `indicators.py`에 12개월 모멘텀 지표 추가 → `scorer.py`에서 하락장에도 상대모멘텀 좋은 종목 선별

### 2.7 리밸런싱 기반 전략 (DCA plus)

**출처**: Ben Graham (1949), "The Intelligent Investor"  
**원리**: 하락할수록 분할 매수 (가치평가 하향 시 비중 확대)

```
KOSPI -5%: 보유 현금의 10% 매수
KOSPI -10%: 보유 현금의 25% 매수  
KOSPI -15%: 보유 현금의 50% 매수
```

**장점**: 심리적 안정, 하락장을 기회로 전환
**구현**: 별도 DCA 모듈 (position_sizing.py)

### 2.8 옵션 기반 전략 (Covered Call)

**출처**: Chicago Board Options Exchange (CBOE)  
**원리**: 보유 주식에 콜옵션 프리미엄 매도로 하락 방어

**한계**: 한국 개인투자자 옵션 거래 제한, 시스템 트레이딩 부적합
