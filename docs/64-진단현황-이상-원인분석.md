| 요소 | 설명 |

**판단**: 하락장(-4.8%)에서 대부분 종목이 기술적 지표 악화는 정상.

### 2.4 3차 원인: 레짐 평가와 실제 시장 괴리

**핵심 모순**: 레짐은 "중립장"으로 평가하지만, 실제 시장은 "급락장" 수준.

| 항목 | 값 |
|:-----|:-----|
| KOSPI 등락률 | **-4.8%** (명백한 급락) |
| 레짐 평가 | **중립장 (41점)** |
| 레짐 bear 기준 | 35점 미만 |
| 스크리너 적용 threshold | 중립장 기준 **30점** |

레짐 점수 41점의 구성:

| 지표 | 점수 | 기간 | 설명 |
|:-----|:----:|:----:|:------|
| MA 정렬 | 55 | 5/20/60일 | 아직 중립 (단기 MA > 장기 MA 유지) |
| RSI | 50 | 14일 | 중립 |
| 볼린저밴드 | 43 | 20일 | 중립 |
| **5일 등락률** | **26** | **5일** | **약세 반영** |
| 거래량 추세 | 45 | 20일 | 중립 |
| ADX | 50 | 14일 | 추세 없음 |

**문제점**:

1. **레짐 평가 지연**: 레짐은 5~20일 데이터 기반. 오늘 -4.8% 급락이 5일 등락률(26점)에만 반영되고, 다른 지표는 아직 중립 유지.
2. **장중보정 한도 부족**: TR-16 B안은 고정 -5점만 보정. KOSPI -4.8% 대비 보정액이 미흡.
3. **임계값 불일치**: 레짐=중립(30점)이 적용되지만, 실제 시장은 약세장 수준이라 42종목 대부분 30점 미달.

→ 스크리너는 "중립장 기준"으로 평가하지만, KOSPI -4.8% 급락장에서는 대부분 종목 점수가 30점 미만이 나오는 것이 정상.

**레짐 보정의 한계**:
```
레짐점수 46 → 장중보정 -5점(고정) → 41점 (중립 유지, bear 35 미달)
```
-5점 고정 보정이 -4.8% 급락을 충분히 반영하지 못함.
등락률 비례 보정(예: 1%당 -1점)이면 46 - 4.8 = 41.2점으로 동일.  
→ -5점 고정 자체는 근접하나, bear 기준(35)에 도달하려면 -11점 필요.

**근본 원인**: 레짐 평가가 5~20일 데이터 기반이라 **당일 급락이 즉시 반영되지 않는 구조적 한계**.

## 3. ML 확률 고정 원인

### 3.1 1차 원인: 단일 종목만 평가

```python
# orchestrator.py L518

```

스크리너 통과가 1개이므로 ML도 1개 종목만 평가.  
**후보 수가 적을수록 ML 예측 다양성도 감소.**

### 3.2 2차 원인: 분봉 데이터 갱신 제한

| 사이클 간격 | 10초 |
|:------------|:------|
| 1분봉 생성 | 60초 = 6사이클 |
| 분봉 캐시 | 사이클마다 초기화 후 재조회 |
| KIS WebSocket | 1분봉 데이터는 **1분 단위**로만 갱신 |

→ 10초마다 ML 추론을 실행하지만, 분봉 데이터는 1분에 한 번만 변경됨.  
→ 결과적으로 **동일 분봉 데이터로 6회 연속 동일 추론** → "고정"처럼 보임.

### 3.3 3차 원인: 하락장에서 낮은 확률

분봉ML 모델(AUC ~68%)이 KOSPI -4.8% 하락장에서 대부분 하락/횡보를 예측.  
확률 2~7%는 **통계적으로 정상** (상승 예측 자체가 드문 시장 상황).
