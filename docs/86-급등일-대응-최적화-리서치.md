# KOSPI 급등일 대응 최적화 리서치 보고서

- 작성일: 2026-07-15
- KOSPI 등락률: +6.8% (폭등)
- 시스템 상태: neutral(54점) / short_term / 매수 0건

---

## 1. 현황 분석

### 병목 체인

```
8개 후보 → MTF(5개 차단) → 3개 → ML(모두 60% 미달) → 0개 매수
```

### 원인 1: MTF 필터 — 추세 전환 첫날 포착 불가

| TF | 상태 | MTF 판정 |
|---|---|---|
| 주봉(장기) | MA5 < MA20, 하락 추세 | ❌ |
| 일봉(중기) | MA5 < MA20, 하락 추세 | ❌ |
| 분봉(단기) | +0.8~1.4% 상승 | ✅ |

- score = 1/3 = 0.33 < neutral threshold(0.67) → **차단**
- 추세 전환일에는 주봉/일봉이 하락인 것이 정상 — MTF가 전환을 인식하지 못함
- 첫날 차단되면 다음날도 같은 패턴 → 영원히 진입 불가

### 원인 2: ML 모델 — 급등 패턴 미학습

ML 예측이 30~41%로 buy threshold(60%)에 크게 미달.
- 모델이 학습한 데이터: 최근 수개월의 하락/횡보장
- 급등 전환 패턴은 학습 데이터에 없음
- ML 모델 재학습(Auto Retrain)은 장 마감 후에만 실행되어 당일 반영 불가

## 2. 수정 방안 (3 Options)

### Option A (추천): 급등일 모드 — 장중 모멘텀 임계값 동적 전환

**아이디어**: 장중 KOSPI 등락률이 특정 임계값(예: +3%)을 초과하면,  
MTF threshold와 ML buy threshold를 일시적으로 낮춤.

| 항목 | 정상 | 급등일 (+3%↑) |
|---|---|---|
| MTF threshold | 0.67 (3TF 중 2) | **0.34 (3TF 중 1)** |
| ML buy threshold | 60% | **50%** |
| position size | 정상 | **1.5x** (자신감 반영) |

**변경 범위:**
- `entry_planner.py` — KOSPI 등락률 체크 후 MTF threshold 동적 전환
- MTF 임계값을 `self._strategy`에서 읽지 않고 상황에 따라 override
- 장중보정 로직(`_apply_intraday_correction`)과 연동

**효과**: KOSPI +6.8%면 MTF 0.34 적용 → 분봉만 상승(1/3=0.33)도 통과.
ML buy threshold 50% → KB금융 30.4%, 삼성화재 41.3%는 여전히 미달이지만,
더 강한 종목(60% 근접)은 통과.

**리스크**: 🟡 급등일 추격 매수로 손실 위험 증가.  
**방어**: min_hold 20분 + stop_loss 1.5%가 하방 방어.

### Option B: MTF bear/neutral 임계값 하향 조정

MTF neutral threshold를 0.67→0.50으로 낮춤 (bear와 동일).
DB config에서 즉시 변경 가능 (`NGSAT_MTF_NEUTRAL_THRESHOLD=0.50`).

**효과**: 분봉만 상승(1/3=0.33)은 여전히 차단.  
분봉+일봉(2/3=0.67)만 되면 통과 → 개선 미미.

### Option C: ML 모델 학습 데이터에 급등 패턴 추가

과거 급등일 데이터를 oversampling하여 ML 모델 재학습.
- 2020년 3월 반등, 2024년 8월 폭등 등 유사 패턴 포함
- 특징: 급등일의 RSI/거래량/변동성 패턴을 피처로 명시적 추가

**효과**: ML 모델이 급등 신호 인식 가능해짐.  
**리스크**: 🔴 대공사. 학습 데이터 수집 + 피처 엔지니어링 + 재학습 파이프라인 수정.

## 4. 즉시 가능한 조치 (DB config)

당장 Option A 구현 전에도 아래 설정 변경으로 일부 개선 가능:

| 설정 | 현재 | 변경 | 효과 |
|---|---|---|---|
| `NGSAT_MTF_NEUTRAL_THRESHOLD` | 0.67 | 0.50 | 분봉+일봉 2/3=0.67 통과 (현재도 차단인 점은 동일) |
| `NGSAT_REGIME_INTRADAY_CAP` | 30 | **40** | 레짐 점수가 +10점 더 올라감 → 54→64, 불장 근접 |
| `NGSAT_BUY_THRESHOLD` | 0.6 | 0.5 | ML 예측 30~41%는 여전히 미달이지만 lower bound 완화 |

**cap=40으로 올리면**: KOSPI +6.8% × 6 = 40.8, cap=40 → 현재 +30보다 +10점 더 반영되어  
레짐 24.3 + 40 = 64.3점 → 불장(65) 직전까지 상승.
