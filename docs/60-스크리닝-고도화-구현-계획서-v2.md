### 2.1 새 파일: `strategy/scorer.py` — 점수 계산 전담

**역할**: 지표 값 → 점수 변환. 레짐별 가중치 관리.

```python
def score_volume(vol_ma5: float, vol_ma20: float, vol_ratio: float, regime: str) -> float:
    ...

def score_candlestick(open, high, low, close, prev_open, prev_close, regime: str) -> float:
    ...

def compute_total_score(scores: dict[str, float], regime: str) -> float:
    """모든 지표 점수를 레짐별 가중치로 합산 + 정규화."""
    weights = get_regime_weights(regime)
    total = sum(scores[k] * weights.get(k, 1.0) for k in scores)
    return max(0, min(100, total / sum(weights.values()) * 100))
```

**변경량**: ~120줄 (신규 파일)

### 2.2 `_evaluate_single_stock()` 경량화

**수정 전**: 80줄 with all scoring inline  
**수정 후**: 30줄 — 지표 계산만 수행, 점수는 `scorer.py`에 위임

```python
def _evaluate_single_stock(stock, prices, threshold, config, index_closes=None):

    mfi_val = last(mfi(highs, lows, closes, volumes, 14))

    # ── 패턴 탐지 (patterns 모듈, 기존 유지) ──
```

## 3. 모듈 구성 (최종)

| 파일 | 역할 | 예상 크기 | SRP 준수 |
|:-----|:------|:---------:|:--------:|
| `strategy/screener.py` | 스크리닝 오케스트레이션 (진입점) | ~250줄 | ✅ 종목 순회/정렬/필터링 |
| `strategy/scorer.py` | **🆕 지표→점수 변환 + 가중치** | ~120줄 | ✅ 점수 계산 전담 |
| `strategy/patterns.py` | 차트 패턴 탐지 | ~400줄 | ✅ 패턴 탐지 전담 (변경 없음) |
| `strategy/indicators.py` | 기술 지표 계산 유틸 | ~330줄 | ✅ 계산 전담 (MFI/OBV/RS/캔들 추가) |
| `core/config.py` | 레짐별 가중치 설정 | +5줄 | ✅ 설정 분리 |

```
screener.py (진입점)
    → indicators.py (지표 계산)
    → scorer.py (점수 변환 + 가중치)    ← 🆕 분리
    → patterns.py (패턴 탐지)
    → core/config.py (설정)
```
