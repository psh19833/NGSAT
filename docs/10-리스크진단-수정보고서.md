## 향후 권장 — Phase 3 (CI 인프라)

```yaml
name: reviewdog
on: [pull_request]
jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: reviewdog/action-flake8@v5
        with:
          flake8_args: --max-line-length=88
```
